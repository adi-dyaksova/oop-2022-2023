## Копи конструктор

```c++
class Number 
{
public:
    int x;
    Number(const Number& other)
    {
        this->x = other.x;
    }
};
```

## Деструктор

```c++
class Number 
{
public:
    int x;
    ~Number()
    {
        std::cout << "The object is destroyed" << std::endl;
    }
};
```

## Оператор =

```c++
class Number 
{
public:
    int x;
    Number& operator=(const Number& other)
    {
        this->x = other.x;
        return *this;
    }
};
```
