## Енкапсулация

```c++
class Number 
{
private:
    int x;

public:
    Number(int n)
    {
        x = n;
    }

    void print()
    {
        std::cout << x;
    }
};
```

## Инициализиращ списък

```c++
class Number 
{
private:
    int x;

public:
    Number(int n): x(n) {}
};
```

## Делегиране на конструктори

```c++
class Number 
{
private:
    int x;

public:
    Number(): Number(0) {}
    Number(int n): x(n) {}
};
```

## Шаблон на клас

```c++
template <typename T>
class Box 
{
private:
    T x;

public:
    Box(T n): x(n) {}
    void print()
    {
        std::cout << x;
    }
};

int main()
{
    Box<int> intBox(3);
    Box<char> charBox('c');
}
```
