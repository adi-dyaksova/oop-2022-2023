## Функциите като тип

```c++
int add5(int x)
{
    return x + 5;
}

int main()
{
    int (*f)(int);
    f = add5;

    std::cout << f(3);

    return 0;
}
```

## Функциите като аргументи на други функции

```c++
int add5(int x)
{
    return x + 5;
}

void printResult(int x, int (*f)(int))
{
    std::cout << f(x);
}

int main()
{
    printResult(3, f);

    return 0;
}
```
